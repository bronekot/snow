# Snowfall Effect

HTML-страница с анимацией снегопада для использования в OBS.

## Использование в OBS

1. Добавьте источник **Browser**
2. В поле **URL** выберите **Local file** и укажите путь к `snow.html`
3. Настройте размер источника (Width/Height)
4. Установите **Custom CSS** (опционально):
   ```css
   body { background: transparent !important; }
   ```

## Настройка

Параметры анимации настраиваются в коде (рекомендую не менять):

- `count` — количество снежинок
- `maxWind` — максимальная сила ветра
